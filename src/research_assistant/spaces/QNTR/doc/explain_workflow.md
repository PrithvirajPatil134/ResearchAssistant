# Explain Workflow Guide - QNTR (Quantitative Research)

> **Workflow**: `explain`
> **Purpose**: Explain quantitative research methods using course materials
> **Last Updated**: 2026-01-31

---

## Expected Input
- `topic`: Quantitative research concept (e.g., "regression analysis", "structural equation modeling")

## Expected Output
- Structured explanation grounded in course slides and readings
- References to Prof. Atul Prashar's class notes
- Statistical methodology with practical application

---

## Grounding Rules

### MUST DO
- [ ] Reference class slides for statistical concepts
- [ ] Cite course readings and methodology papers
- [ ] Include formula/technique explanations
- [ ] Connect to practical research applications

### MUST NOT DO
- [ ] Give textbook answers without course context
- [ ] Skip statistical assumptions discussion
- [ ] Ignore practical limitations

### Priority Source Order
1. Class Slides
2. Prof. Atul Prashar Class Notes
3. Course Outline
4. Key Method Readings
