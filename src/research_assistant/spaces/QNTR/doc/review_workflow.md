# Review Workflow Guide - QNTR (Quantitative Research Methods)

> **Workflow**: `review`
> **Purpose**: Review research methodology submissions against academic standards
> **Last Updated**: 2026-02-02

---

## Expected Input
- `submission_path`: Path to research design, questionnaire, or analysis for review

## Review Criteria (Prof. Atul Prashar)

### Research Design Review
- [ ] Clear research question formulation
- [ ] Appropriate design for question type
- [ ] Theoretical foundation identified
- [ ] Variables properly specified (IV, DV, mediators, moderators)
- [ ] Sampling strategy justified
- [ ] Validity threats addressed

### Questionnaire Review
- [ ] Items aligned to constructs
- [ ] Scale types appropriate (Likert, semantic differential, etc.)
- [ ] Section flow logical (demographics, antecedents, mediators)
- [ ] Item wording clear and unbiased
- [ ] Response options appropriate
- [ ] Length reasonable for target respondents

### Statistical Analysis Review
- [ ] Method appropriate for research question
- [ ] Assumptions checked and reported
- [ ] Sample size adequate
- [ ] Results correctly interpreted
- [ ] Effect sizes reported where relevant
